# 均线法则

</br>

## 有效均线的寻找与切合点原则

</br>

均线周期不是固定的“标准值”，而是一个区间范围，且**每支股票和指数都有其独特的均线参数，而非固定通用参数**。

短期均线一般在 16-25 日之间选择，既能反映短期价格波动，又避免过于敏感产生过多噪音。**最有效的参数通常在 16 到 25 天之间。**

长期均线一般在 45-70 日之间，体现中长期趋势，既不过于滞后，也能有效过滤短期波动。

通过历史数据回测与实盘观察，逐步调整均线周期，**关键是找到最近的“反转点”（无论是向下还是向上）并调整均线以与这些点对齐**。

观察某条均线在过去行情中的支撑和压力表现，找出股价多次反转或确认趋势的均线周期。

结合成交量和 K 线形态，验证该均线是否具有较强的技术参考价值。

不同个股、不同板块甚至不同市场，最适合的均线周期可能不同，需要动态调整。**并非所有公司都存在清晰的均线规则，大约 70%的公司适用。**

</br>

## 多周期均线共振原则

</br>

```bash
选择短期均线（16-25 日）和长期均线（45-70 日）作为主力参考，观察两者的排列和交叉情况。


多周期均线在不同时间框架（如日线和周线）同步发出多头或空头信号时，趋势更可靠。
```

</br>

实战中常见的均线组合示例

</br>

| 均线类型   | 常用周期区间 | 说明                         |
| :--------- | -----------: | :--------------------------- |
| 短期均线   |     16-25 日 | 反映短期趋势，捕捉买卖点     |
| 中长期均线 |     45-70 日 | 判断中长期趋势，支撑和压力位 |
| 辅助均线   |  5 日、10 日 | 辅助短线交易，捕捉快速波动   |

</br>

## 均线切合点的实操建议

</br>

先选定一个大致区间（如 16-25 日），用历史行情回测不同周期的均线表现，找出最能反映趋势和支撑压力的具体天数。

在实盘中观察该均线是否被市场频繁测试且有效支撑或阻力，确认其有效性。在上涨趋势中，如果股价测试并从均线上方反弹，这是买入信号；如果跌破，则是减仓或做空信号。在下跌趋势中，如果股价测试并从均线下方下跌，这是做空或卖出信号；如果带量突破上方，则是买入信号。

结合成交量放大、价格突破等信号，判断均线的有效突破或回踩。

有效均线的核心在于**“切合市场实际，反映真实的支撑压力”**，而非盲目套用固定参数。短期均线以 16-25 日为合理区间，长期均线以 45-70 日为合理区间，通过不断回测和实盘验证，找到最适合当前标的和市场环境的均线周期，才能最大化均线法则的实用性和准确性。

</br>

## 三大趋势

</br>

```bash
连续 3 次被 t 出 直接休息观望
k 线经常穿均线 直接放弃机会 等待
```

</br>

### 多头趋势 （短期 > 中期 > 长期）

</br>

价格沿短期线上扬，形成“均线发散上升”结构。

</br>

价格 ➚  
短均线 ➚  
中均线 ➚  
长均线 ➚

</br>

</br>

### 空头趋势 （短期 < 中期 < 长期）

</br>

价格沿短期线下跌，均线层层压制。

</br>

价格 ➘  
短均线 ➘  
中均线 ➘  
长均线 ➘

</br>

### 盘整趋势 （三线粘合交叉）

</br>

K 线反复穿越均线、方向模糊，需放弃观望

</br>

## 四大法则

</br>

### 分批进场

</br>

```bash
首先想好 本支股票预备投入资金总额 分三笔
```

</br>

#### 最佳买点

</br>

```bash
股价自下往上突破关键点 - x 日均线
```

</br>

##### 有效突破

</br>

```bash
当前为多头趋势（三线向上，短穿长）


突破关键均线后，次日再创新高


均线未被回踩，K线未穿透均线


成交量同步放大，非单日利好暴涨


大盘方向一致，不处于系统性下跌
```

</br>

> 两日法则 (Two-Day Rule):

如果股价突破关键点但第二天未能创出新高，则视为假突破。

如果股价跌破关键点但第二天未能创出新低，则视为假跌破。
这个法则有助于过滤均线信号中的“噪音”，例如假突破或假跌破。

</br>

```bash
0.1%停损 没获利
突破后大红 k 棒 原因是有单一事件影响大涨 这时不进场


指数向下跌破 处于放空位置 第二日有长长下影线 属于假突破 会上升
这时停利 持有 出售 30% 均可
```

</br>

| 问题表现               | 原因排查            | 建议操作            |
| ---------------------- | ------------------- | ------------------- |
| 股价震荡不断穿越均线   | 属于盘整行情        | 放弃等待趋势确认    |
| 进场后频繁被 T 出      | 均线无效/趋势不清晰 | 等有效均线+确认形态 |
| 突破后大涨但无法接回   | 符合“单一事件暴涨”  | 不进场，错过就错过  |
| 突破有效但次日下跌     | 不符合二日战法标准  | 假突破，立即止损    |
| 指数大跌但个股长下影线 | 假跌破（洗盘）      | 可持有或分批止盈    |

</br>

### 快速停损

</br>

```bash
首单 15%-25%资金入场（越低保护本金） 停损点设置 4%-6% 寻找完美停损点
```

</br>

#### 初始投资:

首次“试单”只分配总资金的 15% 到 25%。

正确的利润来自入场那一刻就开始盈利的头寸。

如果投资方向错误，应立即止损。

止损方法应与入场的技术分析方法保持一致（例如，如果基于收盘价跌破 18 日均线止损）。

最佳止损点应在入场前就确定，通常是根据近期低点形成的平行支撑线。

#### 止损设置:

一般止损范围为 4% 到 6%。

通过计算潜在损失来保护总资本，例如，25%的初始投资，7%的止损，总损失为 1.75%。为了将总损失控制在 1%左右，15%的初始投资搭配 7%的止损更为理想（总损失 1.05%）。

### 赚钱加码

</br>

```bash
首单盈利 ≥3%

 均线结构稳定，未出现缠绕/反转

 股价未经历箱体震荡，继续上行

 前期低点已抬升，可设置平行支撑止损
```

</br>

#### 加码方法:

##### 百分比加码法:

如果初始资金为总资本的 15%，后续加码可分为四等份：15%、25%、30%、30%。

如果初始资金为 20%-25%，可分为三等份：如 20%、40%、40%或 25%、40%、35%。

当第一笔盈利达到 3% 时，加入第二批资金。

当第二笔盈利达到 2%-3% 时，加入最后一批资金。

##### 关键点加码法:

当股价有效测试均线并反弹时加码。

当股价在上涨趋势中，经历横盘整理后突破（箱体突破）时加码。

#### 不同股票百分比分批入场法（新概念）:

将总资金分成等份（例如，每份 25%），并在不同的股票突破关键点时分别投资。

止损逻辑保持不变：如果所有已投资股票的总损失超过 1%，则应清仓所有头寸，因为这可能表明对市场状况判断有误。

</br>

### 不接受盈余变亏损

</br>

```bash
第三次加码等待第二笔获利 3%时全部投进去 设置止损为当前成本费金额
（符合第四法则不接受盈余变亏损）
```

</br>

#### 原则:

一旦累积了利润，就应向上调整止损水平。

所有投资都需要止损，例如买入价为 100 元，5%的止损点为 95 元。

如果加码导致总成本提高（例如，从 100 元加码至 103 元，总成本变为 102 元），止损点也必须相应上调。

总损失不应超过总投资的 1%。

一旦进行第三次连续加码，所有账面利润都被视为“既得利润”，股价应继续上涨，不允许出现亏损。

</br>

#### 停利点

</br>

```bash
股价跌破短期均线，卖出50%或全清 即止盈或减仓

 次日再创新低 持仓止盈出清

 跌破长期均线，全部清仓

 获利5% 直接止盈一半 持仓减一半

 单只亏损＞1%，立即全仓止损出局
```

</br>

## 判断「成交量放大」的标准

</br>

```bash
1、日内量比判断（简单粗暴）


✅ 成交量较前一日上涨 2 倍及以上 → 明显放量


✅ 成交量较过去 5 日均量高出 1.5 倍以上 → 有效放量


❌ 只比昨天高一点、仅多出 10%-20%，不构成有效放量，可能是波动噪声


·····················



2、相对均量判断（更精准）


计算过去 N 日（常用 5 日、10 日）平均成交量


当日成交量 ≥ 平均成交量 × 1.5 或 2，才为有效放量


·····················



3、配合 K 线形态


只有「放量阳线」或「放量突破关键点位」才有参考价值。


✅ 放量 + 突破压力位、均线、箱体顶部 → 有效信号


❌ 放量但收阴/十字星/上影线长 → 可能是出货、诱多


·····················


4、排除“单日利好导致暴涨”


特征：


单日成交量暴增 3 倍以上


股价长阳、跳空高开高走


新闻面利好刺激居多（如财报、并购、政策等）


📌 解决方案：


看次日是否继续放量+上涨，或是否快速缩量回落


若次日放量上涨 → 确认有效突破


若次日缩量下跌或回踩 → 说明是假突破
```

</br>

</br>

</br>

### 实盘判断流程总结

</br>

| 判断维度         | 有效放量 | 无效放量 |
| ---------------- | -------- | -------- |
| 较昨日成交量     | ≥ 2 倍   | ≤ 1.2 倍 |
| 较 5/10 日均量   | ≥ 1.5 倍 | ≈ 持平   |
| 是否有关键点突破 | ✅       | ❌       |
| K 线是否阳线     | ✅       | ❌       |
| 是否受消息面驱动 | ❌       | ✅       |

## 有效支撑线

</br>

支撑线是股价下跌过程中，多方反复进场的位置，对价格下跌具有阻止作用。一旦有效支撑，股价容易止跌反弹

</br>

### 1. 历史低点或平台反复验证

</br>

支撑位必须在历史上被反复测试过，至少两次以上不跌破，才称为有效。

支撑位置常见于：

前期低点

整数关口（如 10 元、20 元）

横盘震荡平台底部

示例：股价两次下探 12 元都止住下跌，形成“W 底”，12 元为有效支撑。

</br>

### 2. 价格临近支撑线出现长下影线/K 线反包

</br>

日 K 线呈现：长下影线、锤头线、十字星、早晨之星等止跌形态。

K 线必须出现在支撑区域附近，且当日成交量有放量回升倾向。

</br>

### 3. 放量不破 + 缩量下跌

</br>

下跌至支撑线时缩量，说明抛压减少；

支撑反弹时放量，说明买盘积极。

若支撑时价格“缩量阴跌”，但无明显破位，也属健康。

</br>

### 4. 均线粘合交汇

</br>

多条均线（如 30 日、60 日）在该价位粘合，增强支撑力度。

短期均线向上穿越长期均线（金叉）时，支撑更可靠。

</br>

### 5. 时间验证（回踩不破）

</br>

股价反弹后再次下跌回踩不破原支撑线，就是支撑有效的时间确认。

常见于：突破后的回踩确认支撑。

</br>

假支撑：

无量反弹：支撑后反弹无量 → 容易反复下破。

消息面利空破位：支撑位挡不住重大利空。

支撑被阴线一根大跌穿透，且收盘低于支撑 → 多为有效破位。
