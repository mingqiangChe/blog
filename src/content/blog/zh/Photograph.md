# 摄影

![Perplexity Logo](https://chemingqiang.oss-cn-shenzhen.aliyuncs.com/bag_2/pexels-peterdanthy-32975285.jpg)

## 三要素

<span style="color:#ff5722;font-weight:bold;">光圈（Aperture）</span>
控制进光量与景深

✦ 大光圈（f 值小） ➜ 进光多，景深浅，背景虚化好

适合：<span style="color: #ffc107;">人像、暗光环境</span>

✦ 小光圈（f 值大） ➜ 进光少，景深大，背景清晰

适合：<span style="color: #8bc34a;">风景摄影</span>（建议 F4 - F8 - F10）

📌 <span style="color:#9e9e9e;">光圈影响虚化程度 & 景深范围</span>

<span style="color:#3f51b5;font-weight:bold;">感光度（ISO）</span>
底片对光的敏感程度

✦ ISO 越高 ➜ 照片越亮，但噪点更多，画质下降

不建议高 ISO，除非必须提高亮度（如：夜景）

✦ ISO 越低 ➜ 照片越暗，但画质好，细节清晰

建议：<span style="color:#4caf50;">日常使用 ISO 自动，限制上限</span>

📌 <span style="color:#9e9e9e;">ISO 不宜盲目提高，高 ISO 影响画质与后期空间</span>

<span style="color:#795548;font-weight:bold;">快门速度（Shutter）</span>
决定画面是否清晰 or 拍出动态效果

✦ 高速快门（1/500 秒以上）

清晰定格飞鸟、运动员、溅水等

✦ 低速快门（1/30 秒以下）

拉出车灯、光轨、流水、星轨

✦ 慢门（1 秒及以下）

拍出动态拉影，如水雾绵长、星空旋转

📌 <span style="color:#9e9e9e;">快门配合三脚架更稳定，避免手抖模糊</span>

<br/>

<span style="color:#009688;font-weight:bold;">总结</span>

| 控制项     | 主要作用          | 建议设置示例                |
| ---------- | ----------------- | --------------------------- |
| 光圈       | 控制景深 & 虚化   | 人像 F1.4–2.8 / 风景 F8–11  |
| 感光度 ISO | 控制亮度 & 噪点   | 日光 ISO 100–400 / 夜景限值 |
| 快门       | 控制清晰 & 动态感 | 飞鸟 1/1000 / 星轨 10s+     |

| 环境       | 快门速度         |
| :--------- | ---------------- |
| 静态扫街   | 1/60             |

| 自然风光   | 1/250-1/4        |
| 走动抓拍   | 1/200            |
| 风光拍摄   | 1/30             |
