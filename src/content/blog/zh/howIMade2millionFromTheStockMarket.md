# 第 1 章 与股市有关的历史事实及坊间传言

</br>

## 大众投资热潮与股市崩盘

</br>

核心认知：股市长期向上，但短期波动剧烈，投资者情绪周期性波动影响市场。

</br>

案例：1929 年股市大崩盘前后，投资者因“人人都能发财”理念大量入市，随后遭受惨重损失。

</br>

投资启示：避免盲目跟风，理解市场周期，坚持长期投资更稳健。

</br>

## 长期投资优于债券的事实

</br>

核心认知：即使在大顶买入，长期持有股票仍优于债券和现金资产。

</br>

案例：1929 年顶点买入并持有 30 年，年化收益率达 12.72%，远超债券。

</br>

投资启示：耐心持有优质资产，抵御市场短期波动。

</br>

# 第 2 章 2008 年金融危机：起源、影响及其遗产

</br>

## 次贷危机引发的全球金融动荡

</br>

核心认知：金融衍生品复杂性、评级机构失误及监管缺失导致危机爆发。

</br>

案例：雷曼兄弟破产引发市场恐慌，全球股市大幅下跌。

</br>

投资启示：警惕系统性风险，分散投资和关注宏观经济政策变化。

</br>

## 资产相关性上升与分散化收益下降

</br>

核心认知：危机期间资产间相关性大幅上升，传统分散化策略效果减弱。

</br>

案例：股票、债券、黄金等避险资产均出现不同程度波动。

</br>

投资启示：动态调整资产配置，结合宏观环境调整投资组合。

</br>

# 第 3 章 金融危机后的市场、经济及政府政策

</br>

## 量化宽松与货币政策后果

</br>

核心认知：美联储扩表和低利率政策推动资产价格上涨，但潜藏泡沫风险。

</br>

案例：债券收益率降至零，黄金价格经历暴跌后反弹。

</br>

投资启示：关注货币政策变化，合理配置避险资产。

</br>

## 投资者行为与信任危机

</br>

核心认知：市场信心波动影响资金流向，避险情绪周期性增强。

</br>

案例：危机后投资者偏好安全资产，推动黄金和现金需求。

</br>

投资启示：保持理性，避免情绪化交易，利用市场波动寻找机会。

</br>

# 第 4 章 股票估值与投资策略

</br>

## 估值指标及其局限

</br>

核心认知：市盈率、市净率等估值指标有助判断股票合理价格，但非万能。

</br>

案例：高估值时期买入风险大，低估值时期潜在收益高。

</br>

投资启示：结合基本面和市场环境综合判断估值。

</br>

## 价值投资与成长投资比较

</br>

核心认知：价值投资注重低估资产，成长投资关注未来潜力，两者各有优势。

</br>

案例：不同市场周期中，价值股与成长股表现交替领先。

</br>

投资启示：灵活调整策略，结合个人风险偏好和市场阶段。

</br>

投资启示：掌握风险管理技巧，提升投资安全边际。

# 第 7 章 股票指数：股市的化身

</br>

## 市场大盘 / 道琼斯工业指数 / 市值加权指数

</br>

核心认知：市值加权指数更能反映市场整体趋势，价格加权指数存在局限。

</br>

案例：标普 500 与道琼斯指数长期表现差异明显。

</br>

投资启示：优先选择市值加权指数基金，分散风险。

</br>

## 股票指数的收益率偏差 / 附录：道琼斯原始成份股现状

</br>

核心认知：指数成分股频繁更替，市场结构持续演化。

</br>

案例：道琼斯原始 12 只成分股大多已消失。

</br>

投资启示：指数投资降低个股风险，顺应市场演变。

</br>
